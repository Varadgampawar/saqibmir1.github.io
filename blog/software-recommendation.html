Software Recommendations

Here's a detailed list of all free and open sourse softwares that i use and the ones i recommend.


Software I Use

Operating Systems

GNU/Linux ofcourse. I have used windows 10 and 11 in the past but only for the purpose of gaming. Other that i cant stand using a non unix based operating system. The distro that i use is Arch Linux because of its simplicity and KISS ( keep it simple, stupid!) philosophy.

As far a smart phones are conserned, i have an andorid device on which i use custom roms because most stock roms that come preinstalled are bloated as hell and have a lot of telemetry enabled by default.

I use Lineage OS with open gapps

Shell

I use zsh as my default shell because it supports a lot of plugins and has auto tab completion and syntax highlighting too. You can try my zsh configuration here

Text Editing

I use neovim as my text editor. It is a terminal based vim like text editor with a little more features and plugins. Like vim, it doesn't require mouse and is completely keyboard centric. However, if you wish to use a gui based text editor, i would recommend gedit.

P.S: I don't recommended using bloated IDE's. unless you are working on a huge android application project lol.

Window Manager

I use suckless DWM (dynamic window manager)

I also use other sucless tools like dmenu, st, etc with dwm.

File Manager

I am using ranger which is a terminal based file manager written in python.

Web Browser

My web browser of choice is firefox however i sometimes use brave too because it is chromium based and it has access to all chrome store extensions.

Software I Don't Use

IDE's

IDE's are bloated and are for soydevs. I try to keep my system as minimal as possible

GUI apps

Dont get me wrong. I do use a graphical server. I just prefer not to use a gui front end for a software if there is a cli tool already available for the sake of simplicity. That's the reason why i don't use a desktop environment anymore. The only gui software i have on my system right now is the web browser. Other than that, everything is either a cli tool/terminal based app or handeled by scripts.
